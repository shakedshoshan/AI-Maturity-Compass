# **App Name**: AI Maturity Compass

## Core Features:

- School Details Input: Capture school's name, city, and the user's role to contextualize the assessment.
- Adaptive Questionnaire: Presents a series of questions across key domains (leadership, resources, teaching, etc.) to evaluate AI maturity.
- Scoring Engine: Calculate an overall AI maturity score based on questionnaire responses.
- Maturity Level Assessment: Map the AI maturity score to a maturity level (e.g., Beginner, Intermediate, Advanced) with descriptive labels.
- Radar Chart Visualization: Display a radar chart visualizing the school's performance across different AI domains, highlighting strengths and weaknesses. A tool would compare the results with some ideal target.
- Personalized Recommendations: Provide tailored recommendations based on the school's maturity level and domain-specific scores. This feature would act as a tool.
- Admin Data Aggregation: Aggregate assessment data across multiple schools to provide insights into overall AI maturity trends.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), reflecting the focus on intelligence.
- Background color: Dark navy (#1A237E), creating an immersive atmosphere.
- Accent color: Teal (#00BCD4), providing contrast and highlighting important elements.
- Body and headline font: 'Inter', a sans-serif font, to ensure clarity.
- Use sharp, geometric icons with a filled style.
- Employ a tile-based layout.
- Use subtle, short, non-intrusive animations when transitioning.
